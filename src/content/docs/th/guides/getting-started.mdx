## การติดตั้ง

<Tabs>
  <TabItem label="npm">
    ```bash
    npm install @chanomhub/sdk
    ```
  </TabItem>

  <TabItem label="pnpm">
    ```bash
    pnpm add @chanomhub/sdk
    ```
  </TabItem>

  <TabItem label="yarn">
    ```bash
    yarn add @chanomhub/sdk
    ```
  </TabItem>
</Tabs>

## ขั้นตอนการตั้งค่า

<Steps>
  1. ติดตั้ง SDK ตามวิธีด้านบน
  2. สร้างไฟล์ config
  3. เริ่มต้นใช้งาน
</Steps>

## ตัวอย่างการใช้งาน

```typescript
import { ChanomHub } from '@chanomhub/sdk';

const client = new ChanomHub({
  apiKey: 'your-api-key',
});

// เริ่มใช้งาน
const result = await client.doSomething();
console.log(result);
```

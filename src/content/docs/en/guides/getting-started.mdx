## Installation

<Tabs>
  <TabItem label="npm">
    ```bash
    npm install @chanomhub/sdk
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add @chanomhub/sdk
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add @chanomhub/sdk
    ```
  </TabItem>
</Tabs>

## Setup Steps

<Steps>
1. Install the SDK using one of the methods above
2. Create a config file
3. Start using the SDK
</Steps>

## Usage Example

```typescript
import { ChanomHub } from '@chanomhub/sdk';

const client = new ChanomHub({
  apiKey: 'your-api-key',
});

// Start using
const result = await client.doSomething();
console.log(result);
```

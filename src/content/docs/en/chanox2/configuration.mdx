Guide to customizing and configuring ChanoX2 to suit your needs.

## Accessing Settings

Open Settings in several ways:
- Click the **⚙️** icon in the bottom left corner
- Press `Ctrl + ,` (Windows/Linux) or `Cmd + ,` (macOS)
- Go to **File** → **Settings**

---

## General Settings

### Appearance
| Setting | Description |
|---------|-------------|
| **Theme** | Choose Dark, Light, or System |
| **Accent Color** | Main UI color |
| **Font Size** | Text size (Small, Medium, Large) |
| **Animations** | Enable/disable animations |

### Language
Change the program language:
1. Go to **Settings** → **General** → **Language**
2. Select your desired language
3. The program will restart to apply the new language

### Startup
- **Launch at startup** - Auto-start when booting
- **Start minimized** - Start in minimized mode
- **Check for updates** - Auto-check for updates

---

## Library Settings

### Scan Folders
Manage folders to scan:

```
Settings → Library → Scan Folders
```

- **Add Folder** - Add a new folder
- **Remove** - Remove a folder
- **Exclude** - Exclude subfolders

### Auto Detection
| Option | Description |
|--------|-------------|
| **Detect Engine** | Auto-detect Game Engine |
| **Fetch Metadata** | Fetch game info from the internet |
| **Download Covers** | Auto-download cover images |

### Display Options
- **Default View** - Default view mode (Grid, List, Compact)
- **Sort By** - Default sort order
- **Show Hidden** - Show hidden games

---

## Game Launch Settings

### Windows Compatibility (Linux)
For Linux users running Windows games:

```
Settings → Games → Windows Compatibility
```

| Option | Description |
|--------|-------------|
| **Use Wine** | Enable Wine for Windows games |
| **Wine Version** | Select Wine version |
| **Proton** | Use Proton instead of Wine |
| **DXVK** | Enable DXVK for DirectX |

### Launch Options
Set launch arguments for each game:

1. Right-click game → **Properties**
2. Go to **Launch Options** tab
3. Add arguments such as:
   - `-windowed` - Launch windowed
   - `-skipIntro` - Skip intro
   - `-debug` - Enable debug mode

### Post-Launch Behavior
| Option | Description |
|--------|-------------|
| **Minimize** | Minimize when launching game |
| **Close** | Close when launching game |
| **Keep Open** | Keep open |

## Data & Storage

### Backup
Backup your Library data:

```
Settings → Data → Backup
```

- **Export** - Export data as JSON file
- **Import** - Import from backup file
- **Auto Backup** - Enable automatic backup

### Cache
- **Clear Image Cache** - Clear image cache
- **Clear Metadata Cache** - Clear metadata cache
- **Cache Location** - Change cache location

### Database
- **Rebuild Database** - Rebuild database (fixes issues)
- **Optimize** - Optimize database

## Advanced Settings

### Performance
| Option | Description |
|--------|-------------|
| **Hardware Acceleration** | Use GPU acceleration |
| **Reduce Motion** | Reduce animations |
| **Background Scanning** | Scan while running normally |

### Developer Options
Enable Developer Options:
1. Go to **Settings** → **About**
2. Click version number 7 times
3. Return to Settings to see the **Developer** menu

Developer Options include:
- DevTools access
- Log viewer
- Debug mode
- Experimental features

---

## Reset Settings

### Partial Reset
Go to each Settings section and click **Reset to Default**

### Full Reset
```
Settings → Advanced → Reset All Settings
```

> ⚠️ **Warning**: Reset will clear all settings but won't delete Library data

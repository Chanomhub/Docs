This guide covers how to install ChanoX2 on all platforms.

## System Requirements

| Operating System | Minimum Requirements |
|-----------------|---------------------|
| **Windows** | Windows 10 (64-bit) or higher |
| **macOS** | macOS 10.15 (Catalina) or higher |
| **Linux** | Ubuntu 20.04, Fedora 34, or distros supporting AppImage |

**General Requirements:**
- **RAM**: 4 GB or more (8 GB recommended)
- **Disk Space**: 200 MB for the application
- **Display**: 1280x720 or higher

---

## Windows

### Method 1: Installer (.exe)
1. **Download** the installer from [Downloads](/chanox2)
2. Select `ChanoX2-Setup-x.x.x.exe`
3. Open the file and follow the steps:
   - Click **Next** to start installation
   - Choose installation folder (default recommended)
   - Click **Install**
   - Wait for completion, then click **Finish**

> **Note**: If Windows SmartScreen shows a warning, click "More info" then "Run anyway"

### Method 2: Portable Version
1. Download `ChanoX2-Portable-x.x.x.zip`
2. Extract to your desired folder
3. Open `ChanoX2.exe` to launch

**Portable Version Benefits:**
- No installation required
- Can run from USB drive
- No Registry entries

## macOS

### Installing with DMG
1. **Download** the `.dmg` file from [Downloads](/chanox2)
2. Open the downloaded DMG file
3. Drag **ChanoX2** to the **Applications** folder
4. Launch from Applications or Launchpad

### Gatekeeper Issues
If macOS shows "ChanoX2 can't be opened because it is from an unidentified developer":

1. Open **System Preferences** â†’ **Security & Privacy**
2. In the **General** tab, you'll see a message about ChanoX2
3. Click **Open Anyway**

Or use Terminal:
```bash
xattr -cr /Applications/ChanoX2.app
```

## Linux

ChanoX2 supports multiple installation methods on Linux:

### AppImage (Recommended)
AppImage works on almost all Linux distros without installation:

```bash
# Download and make executable
chmod +x ChanoX2-x.x.x.AppImage

# Run
./ChanoX2-x.x.x.AppImage
```

> **Tip**: Use [AppImageLauncher](https://github.com/TheAssassin/AppImageLauncher) to integrate AppImage with your Desktop menu

### Debian/Ubuntu (.deb)
```bash
# Install with dpkg
sudo dpkg -i chanox2_x.x.x_amd64.deb

# Fix dependencies if needed
sudo apt-get install -f
```

### Fedora/RHEL (.rpm)
```bash
sudo rpm -i chanox2-x.x.x.x86_64.rpm
```

### Flatpak
```bash
# Install from Flathub
flatpak install flathub com.chanomhub.chanox2

# Run
flatpak run com.chanomhub.chanox2
```

## Updating

ChanoX2 supports automatic updates:

1. When a new version is available, you'll see a notification
2. Click **Update Now** to download and install
3. The app will restart after the update completes

**Manual Updates:**
- Download the new version from [Downloads](/chanox2)
- Install over the existing version (your data will be preserved)
